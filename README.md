## MyAssignment.com

### Technology
  * Node Js
  * Express Js
  * MySql
  * Html,Css,Js

### Description
  * It is Assignment Management System Portal Where Student can submit their assignment and see their score.
  * Faculty can assign assignment and also give marks to student.
